# KartingMicrogame
Geta Club Programming Test

The project has 3 scenes that display seven of the eight points required in this test.

1. FreeControlScene
Has a screen menu that allows to test the following:
- Timer mechanic with collectable items adding time to the countdown timer. Selectable form main menu.
- Kart specs modifiers. Two methods implemented: from a controller and from a custom setup (scriptable object). Editable from settings menu.
- Opponents AI with three different difficulties: easy, medium and hard. Editable from settings menu.
- Customizable kart and character. Editable from setting menu.
- Three track trigger mechanics all deployed on track in different sections to test them.

2. LanesControlScene
- Lanes control mechanic. Kart auto-accelerates and the change of lanes is set using the arrows on the keyboard.
- Drift mechanic. It's only enabled on a couple of green wide pads on the ground. Turn to the right direction to enable the drift.

3. Custom scene
- Testing the track prefabs and mounting of a completely new track.
